Sie können unsere interaktive Karte mittels eines <iframe>-Elements in Ihre Artikel einbinden. Nutzen Sie dazu dieses Beispiel:


<iframe width="1170" height="900" frameborder=0 src="https://correctiv.github.io/viz-minijobs-districts-gender/embed/mj-districts-map.html"></iframe>

Passen Sie Breite (width) und Höhe (height) an Ihr CMS / Artikel-Layout an (oder lassen Sie diese Attribute weg und steuern Sie die Dimensionen des iFrames direkt per CSS).

Die Karte passt sich an die Breite des iFrames an, zudem wird für kleinere Screens (Smartphones, Tablets) eine mobil-optimierte Version angezeigt.
